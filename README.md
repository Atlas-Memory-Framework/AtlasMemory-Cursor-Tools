# AtlasMemory Cursor Skills & Agents

Repo-agnostic Cursor skills and sub-agents for planning and execution.

Drag and drop the .cursor file into your own repo.

Feel free to PR with updates & improvements. 

## Usage
1. Use `planning-framework` to create a structured, phased plan.
2. Use `plan-execution` to execute the plan exactly as written.

FYI: These were authored with `5.2-codex` in mind.
